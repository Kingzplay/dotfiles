# dotfiles
My dotfiles that constitute my Linux desktop.

# Requirements
## Programs
- bspwm
- sxhkd
- picom
- dunst
- feh
- xfce4-power-manager
- wmname
- setxkbmap
- polybar
- playerctl
- pactl 
- zscroll
- rofi
- pafix (provided in the repository)

## Fonts and themes
- FantasqueSansMono Nerd Font
- legionlogofont.ttf (custom font provided in the repository)
- Arimo Nerd Font
- FiraCode Nerd Font
- Yaru theme
